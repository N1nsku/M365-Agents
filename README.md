# M365-Agents

## Analyst
- Analyzing structured data, running Python code, generating insights, and visualizing trends.
- Use Analyst when you need to generate a forecast, analyze trends, or extract insights from spreadsheets.

## Researcher
- Gathering background information, summarizing content, identifying patterns, and organizing findings.
- Use Researcher when you want a concise overview, content summary, or structured research on a topic.

## Prompt Coach
- Helping you write clearer, more effective prompts to get better results from Copilot.
- Use Prompt Coach when Copilot's output isn't what you expected, and you want to revise your prompt.

## Idea Coach
- Strengthening and refining your ideas, offering constructive feedback, and stress-testing concepts.
- Use Idea Coach when you want feedback on a proposal or need to improve an idea before sharing it.

## Writing Coach
-  Improving clarity, tone, structure, and grammar in documents, emails, and other written content.
-  Use Writing Coach when you're revising written content for clarity, tone, or grammar.
